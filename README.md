## General info
Android app based on Android Architecture Components (MVVM).
Simple app displaying properties downloaded from API in recycler view. 

## Technology stack
* Minimum SDK level 21
* Jetpack ViewModel & LiveData
* Kotlin 
* Coroutines
* Hilt/Dagger
* Retrofit2
* Glide
